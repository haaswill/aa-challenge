## Getting Started

First, install packages:

```bash
npm install
# or
yarn install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Testing:

Jest:

```bash
npm run test
# or
yarn test
```

Cypress:

```bash
npm run cypress
# or
yarn cypress
```
